## ğŸ“– Overview

**Smart Interactive Resume Builder** is a robust, full-stack application designed to help developers create professional, ATS-friendly resumes with ease. Built with **Next.js 14**, it leverages the power of **Server Actions** for seamless data mutation and **Prisma ORM** for efficient database management.

Unlike static resume generators, this application offers a comprehensive **User Dashboard** where users can manage their projects, skills, and experience centrally. With dynamic theming, real-time previews, and specialized modes (Regular vs. Developer), it provides complete control over how you present your professional identity.
